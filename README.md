# Monorepo virtual chat room

![](https://github.com/sorrowintogold/cat-room-monorepo/tree/main/apps/ui/public/ezgif-5-86ea25567c.gif)

Run development

```sh
pnpm install
pnpm run dev
```

Run tests

```sh
pnpm run test
```

Testing redis connections

```sh
npm i -g redis-cli
rdcli -h <host> -a <password> -p <port>
```
